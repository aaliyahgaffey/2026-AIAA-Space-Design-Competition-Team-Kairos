# 2026 AIAA Space Design Competition: Team Kairos

This repository contains supporting files, analyses, and design tools for Team Kairos' 2026 AIAA Space Design Competition mission concept. The project focuses on a dual-spacecraft heliophysics and communications architecture designed to monitor solar weather, provide early warning for crewed Mars missions, and support Earth--Mars communications.

## Repository Contents

- `kairos_ml.ipynb` — Machine learning model for solar flare classification using SDO data.
- `LinkBudget.xlsx` — Communications link budget calculations.
- `MSOC costing.xlsx` — Mission operations and science operations center cost estimate.
- `Copy of AE443 Kairos Thermal.xlsx` — Thermal subsystem sizing and analysis.
- `Copy of AE443 Kairos Structures Materials...` — Structures and materials analysis.
- `Copy of Kairos Mechanisms Trade Study.pptx` — Mechanisms trade study slides.
- `porkchop_analysis.ipynb` — Trajectory and launch window analysis.

## Note

Large datasets required for the machine learning notebook should be downloaded separately and placed in the same directory as the notebook before running the code.
